A clean, simple and responsive blog theme of WordPress,based on Bootstrap and Font Awesome. Created and maintained by Vtrois.

## install
1.下载最新版 Tale 程序

2.将该项目下载下来之后，将主题文件夹改名为Kratos存放在/templates/themes 文件夹下

3.在后台 -> 系统设置 -> 选择该主题，保存即可
